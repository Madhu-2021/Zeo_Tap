Rule Engine Interface

How to Enter Rules

Create Rule Example

Rule Name: `age_rule`
Rule String: `(age > 30 AND department = 'Sales')`

This rule checks if the user's age is greater than 30 and if the department is 'Sales'.

Evaluate Rule Example

Rule Name: `age_rule`
User Data:
{
"age": 35,
"department": "Sales"
}

This will return True, as both conditions are satisfied.

Complex Rule Example

Rule Name: `salary_experience_rule`
Rule String: `((age > 30 AND department = 'Sales') OR (experience > 5 AND salary > 50000))`

This rule checks if the user is either older than 30 and in the Sales department, or if they have more than 5 years of experience and a salary greater than $50,000.

User Data for Evaluation:
{
"age": 28,
"department": "Marketing",
"experience": 7,
"salary": 60000
}

This will return True because the second condition is satisfied.
